# 📊 ConnectaTel Telecom Usage Analysis

## Project Objective
This project analyzes customer usage patterns in a telecommunications company. 
The goal is to understand how users interact with mobile services such as calls and messages.

## Datasets
The project uses three datasets:

- **plans.csv** – Mobile plans, pricing and included benefits.
- **users_latam.csv** – User demographic information.
- **usage.csv** – Customer activity data including call duration and message length.

## Analysis Steps
1. Data loading and exploration
2. Data cleaning and validation
3. Summary statistics
4. Data visualization
5. Outlier detection
6. Customer segmentation
7. Business insights

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## How to Run
Open the notebook in **Google Colab** or **Jupyter Notebook** and run the cells sequentially.
